## 技术栈

- NestJS
- MongoDB
- Typegoose
- WebSocket (Socket.IO)
- Redis
- GraphQL

## 注意

第一次使用会自动建立用户

密码为: master

可以进入后台管理之后自行修改.

## 补丁

在每次升级之后可能需要打补丁, 使用 `yarn patch` 可以选择可用的补丁
